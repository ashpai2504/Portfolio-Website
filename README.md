# Portfolio Website

Personal portfolio — interactive single-page site (HTML, CSS, JavaScript).

## Run locally

Open `index.html` in a browser, or serve with any static server:

```bash
npx serve .
```

## Contents

- `index.html` — structure
- `styles.css` — styling & animations
- `script.js` — interactions, Three.js viewer, skill wheels
- `Images/` — assets (campus photo, profile picture)
